    - 교통/교통업 (traffic)
    - 음식/요식업 (food)
    - 숙소/숙박업 (accomodation)
    - 관광/관광업 (tourism)

- 개발 환경 
    - OS : Mac OS X, Windows 10 Education
    - 
- Database
    - GitHub Repository
- 배포 
    - Streamlit Sharing

- 과제2 요구조건 충족
    - 출발지 자유 (제주공항 도착 이후)
